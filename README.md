# Sleep Stage Classification Model

This repository contains my Machine Learning final project notebook for sleep stage classification using EEG signals.

## Contents

- `Sleep_Stage_Classification.ipynb`  
  Main Jupyter notebook containing the full project workflow.

- `models/`  
  Saved trained models, including the Random Forest and Compact 1D CNN models.

- `results/`  
  CSV result files generated by the notebook.

- `figures/`  
  Output graphs such as class distribution, feature importance, CNN training curves and confusion matrices.

- `.gitattributes`  
  Git LFS rules for storing large model files.

- `.gitignore`  
  Ignore rules for temporary Python and Jupyter files.

## Project Summary

The project classifies five sleep stages from EEG data:

- Wake
- N1
- N2
- N3
- REM

Two models are included:

1. Random Forest using calculated EEG features
2. Compact 1D CNN using normalized EEG signals

The project focuses on cross-subject classification, where models are tested on people not seen during training.
